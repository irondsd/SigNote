# Tags 🏷️

Tags let you organize notes across all three tiers. A tag is a small colored label — apply it to any combination of [Notes](/docs/notes), [Secrets](/docs/secrets), and [Seals](/docs/seals), then filter by it from search.

---

## ⚠️ Tags are not encrypted

Before anything else, the important part: **tag names are stored as plaintext, even when attached to Secrets and Seals.**

Tags are metadata, like note titles. The server can see them — that's what makes tag filtering work without decrypting anything. The same rule that applies to Secret and Seal titles applies here:

> Never put sensitive information in a tag name. A tag like `binance-password` on an encrypted Secret leaks exactly what the encryption was protecting.

Use tags for categories (`work`, `recipes`, `2026`), not for content.

---

## Adding tags to a note

Open any note and click the tag button in the footer to reveal the tags line, then hit **+**:

- **Search or create in one box** — type to filter your existing tags; if nothing matches exactly, an option to create the tag appears.
- **Most recently used first** — tags you applied recently float to the top of the picker.
- Tags work the same way in the create modals for all three tiers.

Tag names are normalized: lowercased, trimmed, and capped at 50 characters. Each name is unique per account — "creating" an existing tag just selects it.

Note cards show up to 3 tag chips, with a `+N` counter if there are more.

---

## Colors 🎨

Every tag has one of 9 colors (the same palette as note colors). A color is auto-assigned from the tag's name when created, so the same name always gets the same color — and you can change it any time on the tags page.

---

## Managing tags

The **Tags** page lists every tag in your account with a usage count across all three tiers. From there you can:

- **Rename** a tag — the change applies everywhere instantly, since notes reference tags by ID
- **Recolor** a tag
- **Delete** a tag — it's removed from every note that had it; the notes themselves are untouched
- **Click a tag** to jump to search filtered by it

---

## Filtering by tag 🔍

The search page has a tag filter alongside text search, with your most-used tags offered as quick filters. Tag filtering works for all tiers — including Secrets and Seals — because tags are plaintext metadata the server can match without touching the encrypted body.
